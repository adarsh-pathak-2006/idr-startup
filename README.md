# IDR Homepage

A responsive single-page website built for the **Institute of Digital Risk (IDR)** internship assignment.

## Overview

This project presents the mission, services, community, and contact options for IDR through a clean and modern interface using **pure HTML, CSS, and JavaScript**.

## Features

* Responsive layout (mobile & desktop)
* Sticky navigation with smooth scrolling
* Hero section with call-to-action
* Service pillars section
* Train → Hire → Innovate → Deploy pipeline
* Community section
* Contact form with basic validation
* Simple animations and hover effects

## Technologies

* HTML5
* CSS3
* Vanilla JavaScript

index.html

Open **index.html** in a browser.
